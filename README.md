# GaitScan — Frontend

React web app for the GaitScan gait analysis platform.

## Live App
https://gaitscan-frontend-mnt9.vercel.app

## Features
- Video upload with drag-and-drop
- Sport-specific analysis modes (Clinical / Runner / Athlete / Elderly)
- Risk score visualization with color-coded findings
- AI-generated exercise prescription cards
- Session history with trend charts
- Clinician dashboard with patient management
- JWT authentication

## Tech Stack
React · Recharts · Axios
